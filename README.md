# MYSQL-data-cleaning-project
## Overview
Cleaned a real-world layoffs dataset using MySQL, 
identifying and resolving common data quality issues.

## Dataset
Global tech layoffs data containing company, location, 
industry, total laid off, and date information.

## Steps Performed
1. Removed duplicate records using ROW_NUMBER() and CTEs
2. Standardized inconsistent values (industry names, country names)
3. Converted date column from text to DATE format
4. Handled NULL and blank values
5. Dropped unnecessary columns

## Tools Used
- MySQL
- MySQL Workbench
